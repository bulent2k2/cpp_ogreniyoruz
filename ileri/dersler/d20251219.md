Permütasyon Üretimi
----
[İlk yazdığımız uret() işlevi](https://onlinegdb.com/_IASxAepn).     

Dersten sonra biraz daha düzenledim ve programın akışını daha iyi görmek için `derinlik` adlı bir girdi ekledim:

```c++
void üret(S derinlik) {
    if (perm.size() == N) {
        yaz(perm); // permütasyon hazır
        return;
    } 
    for (K k=0; k<N; ++k) {
        if (eklendi[k])      
            continue;        
        eklendi[k] = true;        
        perm.push_back((S) k);       
        // yaz(derinlik, k);        
        üret(derinlik+1);        
        perm.pop_back();        
        eklendi[k] = false;
    }
}    
```

[Kodun hepsi burada](https://onlinegdb.com/ILdUeE0or). `yaz(derinlik, k)`  satırını etkinleştirip çalıştırın. Şöyle bir çıktı olacak:
```
+ k=0  perm: 0
++ k=1  perm: 01
+++ k=2 
         1. permütasyon: 012
++ k=2  perm: 02
+++ k=1 
         2. permütasyon: 021
+ k=1  perm: 1
++ k=0  perm: 10
+++ k=2 
         3. permütasyon: 102
++ k=2  perm: 12
+++ k=0 
         4. permütasyon: 120
+ k=2  perm: 2
++ k=0  perm: 20
+++ k=1 
         5. permütasyon: 201
++ k=1  perm: 21
+++ k=0 
         6. permütasyon: 210
``` 

`+` işareti derinliğe karşılık geliyor. `k` da bizim döngü değişkenimiz, dizinin eleman konumu, yani permütasyona son eklediğimiz sayı.
