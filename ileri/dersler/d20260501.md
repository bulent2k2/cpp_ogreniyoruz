Elif arkadaşınız 47. soruyu paylaştı. Soru şu:
```c
include <stdio.h> // bu satır yok
int main() {
    int ctrl, i = 1, j = 1, k = 1;
    while(j++, i=1, k++) {
        if (abs(k) % (int) 1e8 == 0) printf("%d\n", k);
        if (!k) {
            break;
            ctrl = 0;
        } else if (!i) ctrl = 1;
        else ctrl = 2;
    }
    printf("ctrl=%d", ctrl);
}
```
Yukarıdaki program çalıştığında ekrana ne yazar diye sormuşlar.
a) 0
b) 1
c) 2
d) sonsuz döngüye girer 
e) derleme hatası verir

Önce döngü içindeki `a, b, c` yapısını yani virgül işlemcisini daha iyi anlayalım diye şu kodu yazdık: 
```c
#include <stdio.h>
int f() { return 5; }
int main() {
    int y=0; // komut, değeri ya da çıktısı yok, (statement)
    int x = (1+1, f(), "a", 3+3, 'k', f()); // deyiş ya da terim (expression)
    x++, y = x++;
    printf("x:%d y:%d", x, y);
    return 0;
}
```
Virülle ayrılan terimler ya da deyişler soldan sağa sırayla çalıştırılıyor.
